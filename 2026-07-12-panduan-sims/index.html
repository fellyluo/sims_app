Cara memakai panduan ini

Setiap fitur berisi penjelasan, langkah penggunaan, slot screenshot (kotak putus-putus bila gambar belum diisi), dan storyboard video (klik untuk buka). Peran yang berhak memakai fitur ditandai dengan label hijau.

Mengisi screenshot & video

Gambar diambil dari kotak putus-putus otomatis hilang begitu file gambar tersedia. Letakkan file PNG di folder img/ di samping file ini dengan nama file persis seperti tertulis di tiap slot (mis. dashboard.png). Storyboard di tiap fitur adalah naskah shot-by-shot yang tinggal Anda rekam layar sendiri.

Daftar Nama File Screenshot (Manifest)

Ringkasan seluruh nama file gambar yang dipakai panduan. Simpan semua di folder img/.
